# Qaarib

Qaarib is a Qatar-focused assistant backend for the Fanar Hackathon.

## Run backend

```bash
cd backend
python -m pip install -r requirements.txt
python app.py
```

## Environment

Copy `backend/.env.example` to `backend/.env` and fill in local keys.

## Evaluation

```bash
cd backend
python evaluation/run_eval.py
```
